# 要件定義

## 目的

- お酒の飲み過ぎを防止するためのアプリ

## ターゲット

- 飲み始めてしまうとついつい飲みすぎてしまう人

### 初期ターゲット

- ビールが好きな人

## 方針

小さく作ってリリース、ユーザーの声を聞いて反映

## Poc要件

- ビールの本数（350ml前提）の管理ができればOK
- その日に何本飲むかを入力
- 飲み始める前にボタンをクリックし、飲む本数を記録
- 決めた本数を超える場合は「本当に飲みますか」とダイアログ
  - OKなどクリックした場合は爆弾爆発アニメーション表示、もしくは真っ赤になりUIで警告
- 早くリリースするため、GitHubPagesを利用する
- 後で改修しやすいように素のJavaScriptで乗り切らない
- スマホ利用前提のUIとする

- 冷蔵庫の中からお酒を選ぶようなUIがいい
